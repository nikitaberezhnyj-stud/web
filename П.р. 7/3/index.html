Вертикальний заголовок зліва

Перша колонка

Це перша бічна панель, яка часто використовується для навігації або додаткової інформації. У цьому макеті колонки розташовані вертикально.

Flexbox з column wrap дозволяє створювати вертикальні макети, де елементи можуть переноситися в наступну колонку.

Основний контент

Це головна область контенту сторінки. Саме тут розміщується основна інформація, стаття, пости блогу або інший важливий вміст.

У цьому макеті використовується flex-flow: column wrap, що дозволяє елементам розташовуватися вертикально, але з можливістю переносу в наступну колонку, якщо не вистачає місця по висоті.

Заголовок та підвал тепер виступають як вузькі вертикальні панелі з боків сторінки, створюючи унікальний та цікавий макет.

Давно відомо, що при роботі з макетом читабельний текст заважає зосередитися на елементах макета. Тому використовується достатня кількість тексту для демонстрації.

Третя колонка

Це третя колонка макету, яка зазвичай використовується для додаткової інформації, реклами або пов'язаних посилань.

Вертикальний макет з Flexbox відкриває нові можливості для креативного дизайну веб-сторінок.

Всі колонки мають можливість прокрутки, якщо контент перевищує доступний простір.
